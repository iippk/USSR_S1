# USSR_S 无人自习室智能管理小程序

## 项目概述

**USSR_S** 是一款专为软件工程导论课程设计的无人自习室智能管理小程序，采用微信原生小程序 + 微信云开发技术栈开发，支持多人实时联网使用。

## 技术栈

- **前端**: 微信原生小程序 (WXML + WXSS + JavaScript)
- **后端**: 微信云开发 (云数据库 + 云函数 + 云存储)
- **运行环境**: 微信开发者工具 (Windows最新版)
- **基础库版本**: 2.30.0 及以上

## 云数据库集合

项目使用以下3个集合，已手动创建:

1. **users** - 存储用户信息
   - _openid: 用户唯一标识
   - nickName: 昵称
   - avatarUrl: 头像
   - totalStudyTime: 总自习时长
   - createdAt: 创建时间

2. **seats** - 存储座位信息
   - seatNumber: 座位编号
   - row/col: 行列位置
   - status: 状态 (空闲/已订/使用中/维护中)
   - userId: 当前使用者ID
   - hardwareStatus: 硬件状态

3. **study_records** - 存储自习记录
   - userId: 用户ID
   - seatId: 座位ID
   - duration: 自习时长
   - startTime/endTime: 开始/结束时间

## 云函数列表

1. **login** - 用户登录
2. **reserveSeat** - 座位预约/使用/解锁/硬件控制
3. **getRanking** - 获取排行榜 (总榜/月榜)
4. **getUserStats** - 获取用户统计数据

## 快速开始

### 1. 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号
- 开通微信云开发环境

### 2. 项目配置

1. 在微信开发者工具中导入项目
2. 修改 `miniprogram/app.js` 中的 `env` 为你的云环境ID
3. 修改 `miniprogram/envList.js` 中的 `envId` 为你的云环境ID

### 3. 数据库初始化

在云开发控制台数据库中创建以下集合:
- users
- seats
- study_records

设置权限为: **所有用户可读，仅创建者可写**

### 4. 部署云函数

在微信开发者工具中:
1. 右键 `cloudfunctions/login` 选择 "创建并部署：云端安装依赖"
2. 右键 `cloudfunctions/reserveSeat` 选择 "创建并部署：云端安装依赖"
3. 右键 `cloudfunctions/getRanking` 选择 "创建并部署：云端安装依赖"
4. 右键 `cloudfunctions/getUserStats` 选择 "创建并部署：云端安装依赖"

### 5. 初始化座位数据

首次进入座位页面时，系统会自动初始化座位数据。

## 使用说明

### 首次使用
1. 打开小程序，点击 "立即登录" 进行微信授权
2. 登录成功后，可以在首页查看自习室实时状态
3. 进入 "座位" 页面预约座位
4. 预约后点击座位 "开始使用" 即可开始自习
5. 自习结束后点击 "结束自习" 或到 "学时" 页面结束

### 多人联网测试
1. 使用不同的微信账号登录小程序
2. 同时查看座位页面，座位状态会实时同步
3. 一个用户预约/使用座位后，其他用户会立即看到状态变化
